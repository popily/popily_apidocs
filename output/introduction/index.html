Above is a few rows from a sales database that might be relevant to your users. The data includes dates, names, prices, categories, and locations. With traditional analytics products, turning this data into customer-facing analytics would replace replicated data restructuing and modeling. With Popily, all that is taken care of for you in a few lines of code.

var sourceOptions = {
    connection_string: 'mysql://username:password@host:port/database',
    query: 'SELECT * FROM my_table'
};

popily.api.addSource(sourceOptions, function(err, source) {});

Simple right? Now let's get started creating some analytics.
